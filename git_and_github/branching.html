Git Branching workflow

- Create branch
- Develop new feature
- Commit changes
- Merge feature to main project

Work with branches

- list existing branches
$ git branch

- create new branch
$ git branch branch_name

- switch to branch
$ git checkout branch_name

- merge branch to current brach
$ git merge branch_name

- delete branch
$ git branch -d branch_name
