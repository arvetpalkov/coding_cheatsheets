Git repository workflow

- Working Directory
- Stageing Area
- Repository
Linear work with repository

- Create repository in working directory
$ git init

- Check repository status
$ git status

- Add files to stageing area
$ git add filename_1
$ git add filename_1 filename_2
$ git add .

- Return file from stageing area back to working directory
$ git reset HEAD filename_1

- Check changes in the file
$ git diff filename_1

Save changes to the repository

- Make commit with message
$ git commit -m "Commit message"

- Save changes to previous commit
$ git commit -m "Commit message" --amend
$ git commit --amend --no-edit
