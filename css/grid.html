Create Grid Columns

Create Grid Rows

Grid-template shorthand

Longhand code: 
grid-template-rows: 200px 300px;
grid-template-columns: 20% 10% 70%;
Shorthand code: 
grid-template: 200px 300px / 20% 10% 70%;

Repeat function

grid-template-columns: repeat(3, 100px);
    
grid-template-columns: repeat(5, 1fr);
    
grid-template-columns: repeat(2, 20px 50px);

Minmax function

grid-template-columns: 100px minmax(100px, 500px) 100px;

Grid Item Manipulations

Span Multiple Rows

Longhand code: 
grid-row-start: 1;    /* included */
grid-row-end: 3;    /* excluded */
Shorthand code: 
grid-row: 1 / 3;

Longhand code: 
grid-row-start: 1;
grid-row-end: span 2;
Shorthand code: 
grid-row: 1 / span 2;

Longhand code: 
grid-row-start: span 2;
grid-row-end: 3;
Shorthand code: 
grid-row: span 2 / 3;

Span Multiple Columns

Longhand code: 
grid-column-start: 2;    /* included */
grid-column-end: 5;    /* excluded */
Shorthand code: 
grid-column: 2 / 5;

Longhand code: 
grid-column-start: 2;
grid-column-end: span 3;
Shorthand code: 
grid-column: 2 / span 3;

Grid Area

Longhand code: 
grid-row-start: 2;
grid-column-start: 3;
grid-row-end: 4;
grid-column-end: span 5;
Sorthand-code: 
grid-area: 2 / 3 / 4 / span 5;

Grid Template Areas

.container {
  display: grid;
  grid-template-areas: "a a a"
                       "b c c"
                       "b c c";
}

.a {
  grid-area: a;
}

.b {
  grid-area: b;
}

.c {
  grid-area: c;
}

Overlapping Elements

.container {
  display: grid;
  grid-template: 1fr 1fr 1fr 1fr / 1fr 1fr 1fr 1fr;
}

.a {
  grid-area: 1 / 1 / span 4 / span 2;
}

.b {
  grid-area: 1 / 3 / span 4 / span 2;
}

.c {
  grid-area: 2 / 2 / s+an 2 / span 2;
  z-index: 5;
}
