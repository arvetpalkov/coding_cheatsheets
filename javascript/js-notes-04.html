Accessing Elements

Arrays are zero-indexed. 
  
  fruits[0];    // 'apple'
You can also access individual characters in a string using bracket notation and the index. 

  const myString = 'Hello there.';

  myString[4];    // 'o'
Update Elements 


  fruits[2] = 'mango';
  console.log(fruits);
  // Output: ['apple', 'banana', 'mango']
The .length property 

  fruits.length;    // return: 3
The .push() Method

.push method adds items to the end of an array. 

  fruits.push('kiwi', 'ananas');
The .pop() Method

.pop() removes last item of an array.
.pop() does not take any arguments.
.pop() returns the value of the last element. 
  
  const fruits =['apple', 'banana', 'orange'];

  let fruit = fruits.pop();

  console.log(fruit);     // Logs: orange
  console.log(fruits);    // Logs: ['apple', 'banana']
